Fate/Grand Order est un jeu de rôle au tour par tour avec quelques éléments de roman visuel. Le joueur prend le rôle d'un "maître" et commande un groupe d'individus appelés "serviteurs", qui sont généralement des personnages historiques, littéraires et mythologiques de diverses cultures. Vous pouvez choisir le sexe que vous voulez être et également les nommer.

L'histoire vous a mis dans une installation secrète quelque part dans le monde, nommee Chaldea. Le joueur n'a aucune idée de l'endroit où il se trouve, mais il sait seulement qu'il a été choisi pour remplir uniquement le coq du projet.

Le début de leur voyage est lorsque tout le groupe de Maîtres est mort dans une explosion soudaine, le joueur est allé voir s'il y avait des survivants, un seul. Leur tout premier serviteur, nommé Mash Kyrielight.

Mais avant qu'ils ne rencontrent leur destin, ils ont été envoyés dans le passé grâce à une machine capable de lire le futur et le passé. Dans le prologue, vous avez appris que quelqu'un essaie d'incinérer le monde, ce qui rend impossible d'avoir un avenir en 2018 (2023).
